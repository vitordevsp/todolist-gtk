# Inbox de Planos

Este arquivo recebe notas soltas antes de elas virarem backlog, plano estruturado
ou conhecimento consolidado.

## Notas

- Se uma ideia virar melhoria futura, mova para [`backlog.md`](./backlog.md).
- Se uma frente precisar de varias sessoes, crie uma pasta propria em `docs/plans/`.
- Se uma descoberta tecnica passar a orientar manutencoes futuras, mova para `docs/knowledge/`.
